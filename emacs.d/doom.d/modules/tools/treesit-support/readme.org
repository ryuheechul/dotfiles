#+title: Readme - treesit-support

This module was a way to bridge the gap between [[https://github.com/doomemacs/doomemacs/tree/master/modules/tools/tree-sitter][the old way]] and the lack of support for the new way in doom.

UPDATE: https://github.com/doomemacs/doomemacs/issues/7623 (native =treesit= support in =:tools tree-sitter=) is now closed/resolved upstream, and that module is enabled in =init.el= alongside this one.
