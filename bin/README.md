# bin
Where scripts are

```bash
bin               # scripts for any platform
├── path/         # things that is suppose to be added to $PATH
│   ├── default/    # this will be added to every system
│   ├── wsl/        # only for wsl
│   └── darwin/     # only for darwin
├── darwin        # masOS specicific scripts
└── wsl           # wsl specific scripts
```
